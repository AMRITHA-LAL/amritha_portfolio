About This Project

Smart Portfolio Automation is a Python-based automation project that integrates GitHub API, web scraping, and email automation.


The system automatically fetches GitHub repositories, retrieves the latest news headlines, and sends automated emails using Gmail SMTP.


All outputs are displayed through a live portfolio website hosted using GitHub Pages.
